- React library.

- Styled-components.
- PWA using next-offline.
- SEO using next-seo.

## Getting Started

First, install all dependencies for the project:

```bash
yarn install
```

Second, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.
